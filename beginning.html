Beginning

Build a blog with Areto Framework. Progressive Node.js and user-friendly DBMS MongoDB will help to do this.

Some features

  • Article CRUD
  • Searching
  • Image processing, generation of thumbnails, watermark
  • Article tags
  • Comment management
  • User registration, authentication and authorization
  • Role-based restriction of access to resources
  • Blog administration module
  • Server tasks

The framework source code is available on github.

Required environment

Blog development, using Areto Framework, requires following environment:

  • Server Node.js
  • Database server MongoDB
  • JavaScript IDE - Visual Studio Code (or any similar)

Software versions are available for both Linux and Windows systems.
