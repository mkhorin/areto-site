Admin module

Areto framework makes it easy to develop and manage different parts of an application using modules.

Create module/admin directory in the project root. File structure within a module can be equal to the structure of application files. In fact, a module contains a sub application. A module can have its own configuration, inherit parent components, or create its own.

Append modules section with the admin module in the application configuration. Now all admin actions will be available at admin/{controller}/{action}.
