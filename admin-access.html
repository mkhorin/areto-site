Module access

The areto/filters/AccessControl filter is used in Admin class objects to protect the Admin module from unauthorized access. This filter based on rules defined in rules parameter. Each rule element initializes areto/filters/AccessRule class object.

This allow property defines type of rules allowing (true) or forbidding (false). Permissions property contains an array of access right items that use this rule. In addition to items in rbac/items file, there are two built-in roles:

  • ? - a guest (anonymous) user.
  • @ - a logged-on user.
