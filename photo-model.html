Photo model

The Photo class manages photos of blog frontend. Batch processing of image files are available in the admin module only.

model/Photo.js


        

          The getThumb method returns an address that contains a preview specified size.
          It is used to display photos in view templates:
          
<img src="<%- photo.getThumb(128) %>" alt="<%- photo.getTitle() %>">

          Other methods are shortcuts to the getThumb with different size of images.
