Базовая конфигурация

Создайте в корне проекта директорию config. В ней будут размещаться конфигурационные файлы приложения.

Создайте файл базовой конфигурации config/default.js, где укажите порт для веб-сервера.

config/default.js


        

          Базовая конфигурация может быть расширена файлами,
          одноименными со значением переменной окружения NODE_ENV (development, production).
        

        
Создайте файл config/development.js

        
config/development.js

        

        

          Теперь при старте node с переменной окружения NODE_ENV=development,
          он будет запускать веб-сервер на 3000 порту,
          а при другом значении NODE_ENV на порту 8888 из базовой конфигурации.
