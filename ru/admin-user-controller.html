Контроллер пользователей

Класс UserController отвечает за действия с моделью пользователя в модуле администрирования.

module/admin/controllers/UserController.js


        
        

          Метод actionIndex выводит список всех пользователей с возможностью сортировки
          по идентификатору, имени, email, роли, а также поиском.
